Introdução

Por onde começar?

O objetivo aqui é simples: que você consiga andar só. Os cinco passos abaixo cobrem instalação, acesso a dados e visualização. Domine esses e você já tem base para qualquer análise econômica com R.

Passo 1 · Instalar o R e o RStudio

Minha expectativa é que vocês consigam instalar sozinhos. O vídeo do Código Agro ajuda bastante, mas uma busca por "instalar R e RStudio" no YouTube vai mostrar uma dúzia de opções — qualquer uma serve.

Passo 3 · Base dos Dados

Caso os dados não existam nos repositórios anteriores, ou caso você precise fazer agregações usando SQL, recomendo fortemente o basedosdados.org. Para acessar os dados diretamente pelo R você precisa de três coisas: uma conta no Google Cloud, um projeto válido, e saber acessar os dados.
